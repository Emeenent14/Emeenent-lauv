## Overview

Conduit is an automation platform that simplifies workflow automation for non-technical users. Built on top of n8n, it provides a user-friendly interface with pre-built templates and secure credential management.

## Technical Implementation

### Backend Architecture
- Developed RESTful APIs with Express.js
- Implemented secure credential encryption and storage
- Built integration layer with n8n's workflow engine

### Infrastructure
- Containerized entire stack with Docker Compose
- PostgreSQL for persistent data storage
- Redis for caching and message queuing
- n8n as the automation execution engine

## Why Conduit?

While n8n is powerful, it requires technical knowledge to set up and configure. Conduit bridges this gap by:
- Providing pre-configured templates
- Handling infrastructure complexity
- Offering a simplified, guided UI
- Managing credentials securely

This makes automation accessible to everyone, not just developers.
