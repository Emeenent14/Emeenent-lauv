## Overview

Relay is a cross-platform desktop application designed for managing MCP (Model Context Protocol) servers. Built with Tauri and React, it provides a native experience across Windows, macOS, and Linux while maintaining a small footprint and high performance.

## Technologies Used

- **Rust**: Systems programming for the backend core
- **Tauri**: Cross-platform desktop framework
- **React** + **TypeScript**: Frontend interface
- **SQLite**: Embedded database for local persistence
- **IPC**: Inter-process communication for frontend-backend bridge

## Architecture Highlights

- **Security First**: Tauri's security model limits frontend access to only exposed APIs
- **Performance**: Rust's zero-cost abstractions ensure minimal overhead
- **Offline Ready**: SQLite persistence allows full functionality without network

---

## Why Tauri?

Tauri offers significant advantages over Electron:
- **Smaller binary sizes** (10-20MB vs 150MB+)
- **Lower memory footprint**
- **Better security model**
- **Native OS integration**

This makes Relay fast, secure, and efficient across all platforms.
